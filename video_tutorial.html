Brief description of how the tag works:

<video> tag, in this assignment, consists of 3 attributes:
  • width - decides the width of the video
  • height - decides the height of the video
  • controls - adds the "controls" of the video. ex) play, pause, and volume

Click here to see more information about <video> tag's attributes.

Between the <video> and </video> tags, you can add a <source> tag which brings up the video from the location indicated by "src" attribute.
"type" attribute reads the type of the media; in our case, it's a "video" file formatted in "mp4."
If the browser does not support the video, it will automatically show the texts written between the <video> and </video> tags.

In this webpage, the following example is how the <video> is written in HTML source: 

	<video> width="320" height="180" controls title = "This is a short video named 'under_bridge_1_10.mp4'">
	<source src = "video/under_bridge_1_10.mp4" type="video/mp4">
	Sorry, you browser does not support HTML5/mp4 video.
	</video>
